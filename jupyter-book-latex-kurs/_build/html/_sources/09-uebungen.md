# Übungen

## Übung 1: Start und Struktur

Erstelle ein `article`-Dokument mit:

- Titel, Autor, Datum
- Inhaltsverzeichnis
- mindestens 3 Abschnitten
- einer nummerierten Liste

## Übung 2: Mathematik

Setze:

- eine inline Formel,
- eine nummerierte Gleichung mit `\label`/`\ref`,
- ein `align`-Beispiel über zwei Zeilen.

## Übung 3: Tabelle und Abbildung

Füge ein:

- eine Tabelle mit `booktabs`,
- eine Abbildung mit `\includegraphics`,
- je einen Verweis im Text.

## Übung 4: Literatur

Lege eine Datei `literatur.bib` mit mindestens zwei Einträgen an und:

- zitiere beide Einträge im Text,
- gib das Literaturverzeichnis aus,
- kompiliere vollständig mit `pdflatex` + `biber` + `pdflatex`.

## Übung 5: Mini-Beamer

Erstelle eine 5-Folien-Präsentation mit:

- Titelfolie
- Agenda
- zwei Inhaltsfolien
- Fazitfolie

```{tip}
Arbeite iterativ: Nach jedem kleinen Abschnitt kompilieren und Fehler sofort beheben.
```
